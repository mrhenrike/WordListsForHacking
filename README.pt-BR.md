# WordListsForHacking (pt-BR)

Listas compiladas para pentesters, com foco em palavras comuns de ambientes reais no **Brasil** e outros contextos.

**English:** [README.md](README.md) · [CONTRIBUTING.md](CONTRIBUTING.md) · [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

## Ficheiros principais

| Ficheiro | Descrição |
|----------|-----------|
| `wlist_brasil.lst` | Palavras brasileiras (cultura, música, etc.), 5+ caracteres |
| `username_br.lst` | Nomes de utilizador frequentes |
| `labs_passwords.lst` / `labs_users.lst` | Listas para aulas e eventos (lab) |
| `labs_mikrotik_pass.lst` | Palavras para uso com [MikrotikAPI-BF](https://github.com/mrhenrike/MikrotikAPI-BF) em lab |
